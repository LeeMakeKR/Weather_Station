# 초기 이미지 굽기 및 초기 세팅 가이드 목차

## 라즈베리 파이
1. 이미지 준비
2. 이미지 굽기
	1. 플래싱 프로그램: 라즈베리 파이 이미저
	2. 공식 링크: [Raspberry Pi Imager](https://www.raspberrypi.com/software/)
3. 첫 부팅 전 사전 설정
4. 첫 부팅 후 초기 접속
5. 패키지 업데이트 및 업그레이드 실행
   - 초기 접속 직후 시스템 패키지를 최신 상태로 갱신한다.
   ```bash
   sudo apt update && sudo apt upgrade -y
   ```
6. SPI 활성화 (`raspi-config`)
   - 아래 명령으로 설정 화면에 진입한다.
   ```bash
   sudo raspi-config
   ```
   - `Interface Options` → `SPI` → `Enable` 선택 후 재부팅한다.
   - 재부팅 후 아래 명령으로 SPI 장치 노드를 확인한다.
   ```bash
   ls /dev/spidev*
   ```
7. 전자잉크 구동용 기본 패키지 설치
   - Python 실행 환경과 SPI/Pillow 의존성을 설치한다.
   ```bash
   sudo apt install -y git python3-pip python3-pil python3-spidev
   ```
8. Waveshare 드라이버(예제 코드) 설치
   - 공식 저장소를 내려받아 보유한 패널 모델 예제를 실행할 준비를 한다.
   ```bash
   git clone https://github.com/waveshareteam/e-Paper.git
   ```
9. 예제 실행 준비 확인
   - 예제 디렉터리로 이동해 패널 모델에 맞는 스크립트를 확인한다.
   ```bash
   cd e-Paper/RaspberryPi_JetsonNano/python/examples
   python3 epd_4in2_V2_test.py
   ```


## 오렌지 파이
1. 이미지 준비 http://www.orangepi.org/html/serviceAndSupport/index.html
2. 이미지 굽기
	2. 발레나 에처 공식 링크: [balenaEtcher](https://etcher.balena.io/)
3. 첫 부팅 전 사전 설정
4. 첫 부팅 후 초기 접속
   > ⚠️ 오렌지 파이는 초기 WiFi/SSH 설정이 되어 있지 않아, USB 또는 HDMI로 직접 접속한 후 수동으로 설정해야 한다.
   - 아래 명령으로 오렌지 파이 설정 화면에 진입한다.
   ```bash
   sudo orangepi-config
   ```
5. SPI 활성화
   - `orangepi-config` → System → Hardware 에서 `spi0-spidev` 항목을 활성화한 후 저장하고 재부팅한다.
   - 재부팅 후 'ls /dev/spi*' 명령으로 `/dev/spidev0.0` 장치가 생성되었는지 확인한다.
   ```bash
   ls /dev/spi*
   ```
6. 패키지 업데이트 및 업그레이드
   - 초기 접속 후 반드시 아래 명령을 실행하여 시스템 패키지를 최신 상태로 유지한다.
   ```bash
   sudo apt update && sudo apt upgrade -y
   ```
7. 전자잉크 구동용 기본 패키지 설치
   - Python 실행 환경과 SPI/Pillow 의존성을 설치한다.
   ```bash
   sudo apt install -y git python3-pip python3-pil python3-spidev
   ```
8. Waveshare 드라이버(예제 코드) 설치
   - 공식 저장소를 내려받아 보유한 패널 모델에 맞는 예제를 실행할 준비를 한다.
   ```bash
   git clone https://github.com/waveshareteam/e-Paper.git
   ```
9. Orange Pi 보드 감지 분기 수정 (필수)
   - 기본 `epdconfig.py`는 Orange Pi를 Jetson으로 오인해 `Jetson.GPIO` 오류가 발생할 수 있다.
   - `OrangePi` 모델 문자열도 Raspberry 계열 분기로 인식하도록 조건을 수정한다.
   ```bash
   cd ~/e-Paper/RaspberryPi_JetsonNano/python/lib/waveshare_epd
   cp epdconfig.py epdconfig.py.bak
   sed -i 's/"Orange Pi" in model/"OrangePi" in model or "Orange Pi" in model/' epdconfig.py
   ```
10. 추가 의존성 설치 (gpiozero)
   - Raspberry 분기로 동작 시 `gpiozero` 모듈이 필요하므로 설치한다.
   ```bash
   sudo apt install -y python3-gpiozero
   ```
11. 전자잉크 테스트 예제 실행 (4인치 모델)
   - examples 폴더에서 4인치 예제를 확인한 뒤, 파일을 하나씩 실행해 패널 호환 모델을 찾는다.
   ```bash
   cd e-Paper/RaspberryPi_JetsonNano/python/examples
   ls -1 epd_4in*.py
   python3 epd_4in2_test.py
   ```
12. 화면 표시 확인 및 마무리
   - Waveshare 테스트 이미지(또는 텍스트)가 정상 표시되면 배선/SPI/드라이버 설치가 완료된 상태다.
   - 실행 중 동일한 예외가 반복되면 `epdconfig.py.bak`과 현재 파일을 비교해 분기 수정이 반영되었는지 먼저 점검한다.
   - 필요 시 종료 후 슬립 모드 처리 루틴을 적용해 잔상과 전력 소모를 줄인다.
