## 4. 전자잉크 배선 참고 (이미지 기준)

| 전자잉크 핀 번호/이름 | 라즈베리 파이 제로 2W 핀 번호 (Physical) | GPIO 번호 |
| --- | --- | --- |
| VCC | 1번 | 3.3V (권장) |
| GND | 6번 | GND |
| DIN (MOSI) | 19번 | GPIO 10 |
| CLK (SCK) | 23번 | GPIO 11 |
| CS | 24번 | GPIO 8 |
| DC | 22번 | GPIO 25 |
| RST | 11번 | GPIO 17 |
| BUSY | 18번 | GPIO 24 |

참고: 실제 배선은 사용 중인 전자잉크 모듈의 데이터시트와 함께 최종 확인하세요.

## 5. Waveshare 전자잉크 드라이버 설치 기록 (RPI2W)

기록 일시: 2026-06-22

### 5.1 SSH 접속

```bash
ssh raspberry@172.30.1.51
```

접속 확인:
- 사용자: `raspberry`
- 호스트: `RPI2W`

### 5.2 SPI 활성화 상태 확인

```bash
sudo raspi-config nonint get_spi
```

결과:
- `0` (SPI 활성화 상태)
