## 4. OS 설정
1. SPI 활성화
   - `raspi-config` → Interfacing Options → SPI → Yes 로 활성화 후 재부팅. `/dev/spidev0.0` 장치가 생성되어야 정상.
2. GPIO 접근 권한
   - 일반 사용자 계정으로 GPIO/SPI 장치에 접근하려면 `gpio`, `spi` 그룹에 추가해야 한다. (`sudo usermod -aG spi,gpio $USER`)
3. 부팅 시 모듈 로드
   - `/etc/modules` 또는 `/boot/config.txt`에 `spi-bcm2835` 등 필요한 커널 모듈을 등록해 재부팅 후에도 자동으로 로드되게 설정한다.
